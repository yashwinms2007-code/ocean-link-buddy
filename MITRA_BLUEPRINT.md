# Mitra Application Blueprint & Lovable Prompt

🚀 **COPY THIS PROMPT FOR LOVABLE**

Create a modern, mobile-first web application named “Mitra”, designed for fishermen safety, smart navigation, and fish trading.

The app should have a clean ocean-themed UI (blue/white), simple navigation with icons, and be fast, lightweight, and easy to use.

---

## 🌐 Multi-language
- Support **English**, **Kannada (ಕನ್ನಡ)**, and **Hindi (हिंदी)**.
- All features including voice alerts and chatbot must work based on the selected language.

---

## 🔐 Authentication
- Login using phone number (OTP simulation).
- Redirect to dashboard after successful login.

---

## 👤 Profile Page
- Show app name **Mitra** with fish + wave logo.
- Display user name and phone number.
- Edit profile option.
- Language switch (EN / ಕನ್ನಡ / हिंदी).

---

## 🏠 Dashboard
Include high-contrast icons for:
- **Fish Detection**: Sonar-style UI with AI predictions.
- **Weather**: Real-time maritime reports with color-coded alerts.
- **Sea Map**: Live GPS tracking and safety zones.
- **SOS Emergency**: High-impact red button for distress signals.
- **Safety**: Educational tutorials with voice guidance.
- **Fish Market**: Buyer/Seller marketplace for fresh catch.
- **Notifications**: Critical alerts and community updates.
- **Settings**: App preferences and emergency contacts.

---

## 🤖 AI Chatbot (Voice & Text)
- Dedicated chatbot named Mitra.
- Support for voice commands and text queries.
- Context-aware responses based on marine safety, weather, and fishing data.

---

## 🛠️ Technical Requirements
- **Fast & Lightweight**: Minimal dependencies for low-bandwidth maritime use.
- **Routing**: standard React Router navigation (no lazy loading for critical stability).
- **Styling**: Standard Tailwind CSS with a blue/white ocean palette.
- **State Management**: Persist language choice in localStorage.
